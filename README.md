# Jsonnetic

CLI Tool based on [jsonnet](https://jsonnet.org), but with a few extra native functions.

# Build Jsonnetic

```sh
git clone git@github.com:neticdk/go-jsonnetic.git
```

```sh
make build
```

Clean slate builds:

```sh
go clean --modcache
go get -u
go build
```
