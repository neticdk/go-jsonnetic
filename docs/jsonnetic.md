## jsonnetic

Jsonnetic is a jsonnet implementation with a extra native functions

### Synopsis

Jsonnetic adds the following native functions to jsonnet:

- fileExists
- regexMatch
- regexSubst
- promqlAggregateBy


```
jsonnetic [command] [flags]
```
